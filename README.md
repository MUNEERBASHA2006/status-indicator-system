## Components
- Arduino Uno
- LED
- 220 ohm resistor
- Jumper wires

## Wiring
- Pin 13 → 220 ohm resistor → LED long leg
- LED short leg → GND

## Real World Application
Used in routers, chargers, and electronic devices to indicate power or status.
